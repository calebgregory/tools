When working with any voice-memo transcript — whether for a daily note, ad-hoc transcription, or any other context — apply known corrections from `transcription-corrections.md` directly to the raw transcript text, not only to a summary/outline derived from it.

**Why:** The user confirmed they want corrections in the raw transcript as well. The "raw" transcript exists as a faithful record of *what was said*, not *what the transcriber heard* — known mistranslations should be fixed in place. Preserving transcriber errors verbatim has no value.

**How to apply:** When writing or updating any document that contains a transcript, scan the transcript text for entries in `transcription-corrections.md` and replace them inline. Do this in the same pass as any summary/outline work. No need to flag the corrections in the response — just apply them.
